# OpenWeb

Developing iOS, Android, and React Native SDKs for the **#1 conversation tool**, with customizable UIs and flexible publisher integration.

## The Challenge

OpenWeb needed cross-platform SDKs that would allow publishers to embed rich, real-time conversation experiences into their apps while maintaining full UI customization and brand consistency.

## Our Approach

- Developed native **iOS**, **Android**, and **React Native** SDKs in parallel
- Implemented **MVVM architecture** for clean separation of concerns
- Built a customizable UI system allowing publishers to match their brand identity
- Created a flexible integration layer supporting diverse publisher tech stacks

## Tech Stack

- React Native
- iOS (Swift)
- Android (Kotlin)
- MVVM Architecture

## Results

- Powers the **#1 conversation platform** on the web
- Adopted by major publishers worldwide
- Fully customizable UI across all platforms
