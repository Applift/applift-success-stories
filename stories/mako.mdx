## About

12+ is Israel's biggest streaming platform. Channel 12 live broadcasts, news, reality, comedy, drama on demand. Mako, Keshet Media Group's digital side, pulls in over 20M views a month across web and connected TV.

Smart TV native work is hard in ways that web work isn't. Samsung Tizen, LG webOS, and Hisense VIDAA each run their own runtime. Memory budgets are tight. The hardware is unforgiving if you cut corners. Mako needed more developers in that environment without slowing down the apps already in production.
