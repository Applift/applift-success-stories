<Overview>
Unit is the banking-as-a-service platform that lets companies embed powerful financial features into their product. We develop multiple native mobile SDKs used by Unit's clients, author implementation guides, and support client integrations. For example, we built a Mobile SDK for adding Unit credit cards to Apple Wallet, Google Pay, and Samsung Pay using Visa's push provisioning.
</Overview>
