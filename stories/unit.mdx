## About

Unit is the banking-as-a-service platform that lets companies embed financial features directly into their products. Its white-label UIs provide pre-built, customizable components that let Unit's clients build in minutes and launch in weeks.

## Native SDK Engineering

We are the mobile team behind Unit's client-facing SDKs, working in lockstep with Unit's internal web team. Applift owns the native mobile surface area end-to-end: architecture, development, and ongoing evolution. These SDKs are shipped inside the production apps of Unit's clients.

* **Architecture & Development:** We lead the technical architecture and implementation of the iOS SDK in Swift and the Android SDK in Kotlin, with a focus on a lightweight footprint, strict API boundaries, and the reliability required for financial services.
* **Digital Wallet Provisioning:** We built push-provisioning flows that issue cards directly into **Apple Wallet**, **Google Pay**, and **Samsung Pay** via **Visa push provisioning**, handling the platform-specific cryptography, entitlements, and certification requirements each wallet demands.
* **DevOps & Distribution:** We own the full release pipeline, from automated testing and versioning to binary distribution, so Unit's clients receive stable, well-tagged releases they can integrate with confidence.
* **Developer Experience:** We treat the SDKs as a product for developers. Public APIs, error surfaces, and sample code are designed for fast, unambiguous integration into client applications.

## Cross-Platform SDK (React Native)

Alongside the native work, we are building a React Native SDK that wraps the underlying iOS and Android implementations. Cross-platform teams get the full native feature set through an idiomatic React Native surface, without us forking the business logic across stacks.

## Partnering with Unit's Clients

Our scope extends beyond building the SDKs. For each SDK, we write the implementation guide that Unit's clients use to integrate, and we work directly with those client teams during the integration itself. When a client hits a platform edge case or a question the documentation doesn't cover, we are the engineers on the other end. In practice, we operate as an extension of Unit's platform team for downstream developer enablement.

## The Ongoing Partnership

The engagement is live and expanding, with new capabilities landing across all three SDKs as Unit's platform grows and new clients come online.
