# Unit

Unit is the banking-as-a-service platform that lets you embed powerful financial features into your product. Build in minutes, launch in weeks.

We are developing multiple native mobile SDKs that are being used by Unit's clients. In addition to writing an implementation guide for each SDK, we also help Unit's clients with the integration process.

For example, we developed a Mobile SDK that allows Unit credit cards to be added to Apple Wallet, Google Pay or Samsung Pay. The SDK is using Visa's push provisioning SDK.

## Engineering Focus

- Native Mobile SDK Development: Built multiple production-grade mobile SDKs enabling Unit's clients to embed banking features directly into their own applications.
- Digital Wallet Integration: Developed a dedicated SDK for adding Unit credit cards to Apple Wallet, Google Pay, and Samsung Pay using Visa's push provisioning technology.
- Client Integration Support: Authored comprehensive implementation guides for each SDK and provided hands-on support to Unit's clients throughout the integration process.
- Cross-Platform Delivery: Ensured consistent SDK behavior and reliability across iOS and Android platforms in a highly regulated financial environment.

## Technical Stack

- **Platform**: iOS, Android
- **Integrations**: Apple Wallet, Google Pay, Samsung Pay
- **Provisioning**: Visa Push Provisioning SDK

## The Impact

Our ongoing partnership with Unit has resulted in a suite of robust, client-facing mobile SDKs that power the financial experiences of their growing customer base. By handling the complexity of digital wallet provisioning and providing white-glove integration support, we help Unit deliver a seamless banking-as-a-service experience to the market.
