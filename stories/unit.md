# Unit

Contributing to the **Banking-as-a-Service** platform that lets businesses integrate banking, lending, and payment services via API.

## The Challenge

Unit needed experienced developers to accelerate feature delivery on their BaaS platform, which enables companies to embed financial services directly into their products through a robust API layer.

## Our Approach

- Contributed to core platform features using **TypeScript** and **React**
- Worked closely with the product team on API design and developer experience
- Implemented new banking and payment service integrations
- Maintained high code quality standards in a regulated financial environment

## Tech Stack

- TypeScript
- React
- REST APIs

## Results

- Accelerated delivery of the **BaaS platform**
- Improved developer experience for API consumers
- Maintained compliance with financial regulations
