## About

Outbrain is a leading recommendation platform for the open web. Its technology enables one-third of the world's online consumers to discover new things through recommendation feeds on their favorite media, and connects advertisers to those audiences.

## Smartfeed Mobile SDK

The Smartfeed Mobile SDK is the mobile surface of Outbrain's recommendation platform. It fetches and displays personalized content recommendations directly within the content of publisher apps, reaching readers inside titles like CNN, Sky News, ESPN, and many more.

Our work spans all three client platforms - iOS, Android, and React Native - with a consistent API and behavior across each, so publishers get the same integration experience regardless of the stack they ship on.

A full CI release pipeline publishes tagged, verified builds to the package managers publishers integrate with - CocoaPods and Swift Package Manager for iOS, Maven for Android, and NPM for React Native.
